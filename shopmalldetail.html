Service Info.

Main page

1) MySQL을 이용해 상품 table 생성 후 node.js와 연결해 'more view' 클릭 시 해당 데이터 가져오기

2) tab기능을 이용해 type별 상품 노출

3) 상품별 호버 시 해당 상품 이름 모달창 노출

4) 상품 클릭 시 해당 상품 상세페이지 이동

Detail page

1) redux(createSlice)로 상태관리

2) 구매하기 클릭 시 바로 장바구니로 이동되며 해당 상품 수량 +1

3) 장바구니 담기 클릭 시 이중 모달창 : 확인 모달창 노출 -> 확인 클릭 시 장바구니로 이동되며 수량 +1

4) 장바구니에 중복 상품 있을 경우 수량 +1, 없을 경우 항목 추가

5) 탭 기능 구현 : useState로 상태관리 (+애니메이션 효과)

6) 하단 탭메뉴1 : 공간 및 배경색 클릭시 해당 이미지와 배경색상 노출

7) 하단 탭메뉴2 : 각 텍스트 아코디언 메뉴로 노출

Cart page

1) redux(createSlice)로 상태관리

2) 각 상품별 수량에 따른 가격변동과 총 합계 금액 계산

3) 각 상품별 삭제버튼 클릭시 -> 확인 모달창 노출 -> 확인 클릭시 삭제

4) 수량 변경 버튼 + 수량이 음수가 되지 않도록 alert 창 노출
