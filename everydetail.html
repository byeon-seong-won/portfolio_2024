OverView.

※ swiper(horizontal) + circle progress.js + css animation

전체 페이지를 Swiper 라이브러리를 이용해 섹션 별로 부드럽게 전환될 수 있도록 풀페이지 효과를 주었습니다. 섹션 전환 시 circle progress bar와 애니메이션 효과, swiper 효과가 실행되도록 스크립트를 사용하였고, 반응형일 경우 scrollTop 값과 브라우저 창의 가로 값과 세로 값을 이용해 변경될 수 있도록 작업하였습니다. 또한 접근성을 위해 다크 모드로 설정되었을 경우를 고려하여 색상에 반전을 주며 가독성에 유의하였습니다.

Service Info.

01. section 01
- css 애니메이션을 이용해 background가 점점 커지는 애니메이션 구현
- aos 라이브러리를 이용해 이미지가 fade-in되는 애니메이션 구현
- swiper 슬라이드의 slideChange 메서드를 이용해 activeIndex에 들어올 경우 addclass를 이용해 애니메이션을 실행하도록 구현
- 해당 activeIndex에 벗어난 경우 removeClass를 이용해 애니메이션을 초기화

02. section 02
- circle progress 라이브러리를 이용해 전환되는 시점에 progress bar를 특정 범위까지 채우는 애니메이션 구현
- swiper 슬라이드의 slideChange 메서드를 이용해 activeIndex에 들어올 경우 해당 circle progress 스크립트가 실행되도록 구현

03. section 03
- css애니메이션을 이용해 손 이미지가 아이콘을 클릭시 동전이 움직이고, 숫자가 "503" 까지 올라가도록 애니메이션 구현
- 슬라이드 전환시 숫자는 "500"으로 초기화되도록 구현
- swiper 슬라이드의 slideChange 메서드를 이용해 activeIndex에 들어올 경우 addclass를 이용해 애니메이션을 실행하도록 구현
- 해당 activeIndex에 벗어난 경우 removeClass를 이용해 애니메이션을 초기화

04. section 04
- swiper를 이용해 브랜드 이미지가 자동으로 슬라이드 되도록 구현
